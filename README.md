# BRINGBACKSUMMER
BRINGBACKSUMMER, mobil arayüzü React Native ve Expo ile geliştirilen, web sitesi next.js ile geliştirilen, backend sunucusu node.js & express rest api ile geliştirilen otel bulma uygulamasıdır.
